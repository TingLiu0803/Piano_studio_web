# Comprehensive SEO Audit Checklist

## 1. Technical SEO (Beyond Lighthouse)

### Core Web Vitals
- [ ] Page load speed < 2.5s
- [ ] Largest Contentful Paint (LCP) < 2.5s
- [ ] First Input Delay (FID) < 100ms
- [ ] Cumulative Layout Shift (CLS) < 0.1

### Crawlability
- [ ] robots.txt properly configured
- [ ] sitemap.xml submitted to Google Search Console
- [ ] All pages return 200 status codes
- [ ] No broken internal links
- [ ] Proper 301 redirects (no 404s)

### Mobile Optimization
- [ ] Mobile-friendly test passes
- [ ] Touch targets are 48x48px minimum
- [ ] Text is readable without zooming
- [ ] No horizontal scrolling

## 2. On-Page SEO (Keyword Targeting)

### Title Tags
- [ ] Each page has unique, keyword-rich title (50-60 chars)
- [ ] Primary keyword in first 60 characters
- [ ] Location keywords included (San Jose, South Bay)
- [ ] Brand name included
- [ ] No keyword stuffing

### Meta Descriptions
- [ ] Unique descriptions for each page (150-160 chars)
- [ ] Includes primary keyword naturally
- [ ] Includes call-to-action
- [ ] Compelling enough to improve CTR

### Headers (H1-H6)
- [ ] One H1 per page with primary keyword
- [ ] H2s support main topic with related keywords
- [ ] Proper header hierarchy (H1 → H2 → H3)
- [ ] Keywords in headers are natural, not forced

### Content Quality
- [ ] Minimum 300 words per page (ideally 500+)
- [ ] Primary keyword density 1-2%
- [ ] Related keywords naturally included
- [ ] Content answers user intent
- [ ] Unique content (not duplicated)

### Internal Linking
- [ ] Important pages linked from homepage
- [ ] Related pages linked contextually
- [ ] Anchor text is descriptive (not just "click here")
- [ ] Links to trial/contact pages from every page
